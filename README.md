# product-web

Project build with:

- [Angular 8 CLI](https://cli.angular.io/)
- [Angular Material](https://material.angular.io/) for Components

Running in `http://localhost:4200/`

## Requirements

- [Docker](https://docs.docker.com/engine/install/ubuntu/) >= 19.03.8

## Build

Execute `make build`

## Running

Execute `make up`
